# Changelog

All notable changes to this project are documented in this file.

## [1.0.3] - 2026-02-16

### Added

- ForceFlow branding updates for app and widget names
- Post-save widget reminder flow when widget is not yet added
- Launcher icon color variants with alias-based switching
- Repository documentation (`README.md`) and license

### Changed

- Android package name updated to `com.toutakun04.forceflow`
- Widget rendering now follows selected theme color more consistently
- Save flow now updates widget data/color immediately after successful fetch

### Fixed

- Handle/API error feedback messaging in config screen
- Launcher alias stability issues that caused launch component errors
- Sync timing issues between saved theme color and widget redraw
