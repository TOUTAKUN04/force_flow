# ForceFlow

ForceFlow is an Android home-screen widget app for Codeforces statistics.

## Features

- Codeforces handle setup and quick validation
- Widget heatmap (last 365 days)
- Solved count, streaks, rating, rank, and last solved problem
- Theme color customization for widget visuals
- Dynamic launcher icon color variants

## Tech Stack

- Kotlin
- Jetpack Compose
- Glance App Widget
- WorkManager
- Retrofit + Gson

## Package

`com.toutakun04.forceflow`

## Build

1. Open the project in Android Studio.
2. Sync Gradle.
3. Run the app on an emulator or Android device.
